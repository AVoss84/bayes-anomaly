Package installation
====================

.. contents::
   :local:

Install *bhad* package using `uv <https://github.com/astral-sh/uv>`__:

.. code-block:: bash

   uv venv --python 3.12         
   source .venv/bin/activate
   uv add bhad

Alternatively, using pip:

.. code-block:: bash

   python3 -m venv .venv
   source .venv/bin/activate
   pip install bhad
